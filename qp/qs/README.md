# QS Software Tracing Component

Software tracing implementation for QP framework. See [documentation](../../docs/qs/README.md).

## Features
- 100 predefined record types
- HDLC framing with byte-stuffing
- Dual 128-bit filtering
- Dictionary support (5 types)
- 35 trace macros
